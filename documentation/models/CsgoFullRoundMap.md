# CsgoFullRoundMap

The location selected during the picks and bans phase for the game.

**Properties**

| Name      | Type | Required | Description                    |
| :-------- | :--- | :------- | :----------------------------- |
| id\_      | int  | ✅       | The ID of the map.             |
| image_url | str  | ✅       | A URL to the image of the map. |
| name      | str  | ✅       | The name of the map.           |
